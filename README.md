# Viccf plugin

This is a contact form plugin for wordpress.

# Installation

Clone plugin into your plugins folder

To install npm run:
```bash
npm install
```
or
```bash
yarn
```

To start watch run
```bash
npm start
```
or 
```bash
yarn start
```

To build run:
```
npm run build
```
or
```
yarn run build
```

For more info visit [author's site](https://mithamovictor.github.io)
